# Caribe Delivery 📦

Caribe Delivery es una landing page desarrollada como proyecto personal con el objetivo de presentar un servicio de entregas para República Dominicana mediante una interfaz moderna, intuitiva y adaptable a diferentes dispositivos.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Características

- Diseño responsive para computadoras y dispositivos móviles.
- Interfaz limpia y moderna.
- Navegación sencilla.
- Organización del código por archivos.
